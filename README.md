# CAN Bus Communication Network

This project demonstrates a CAN Bus communication network using Arduino Uno and Arduino Nano boards.

## Features
- Multi-node CAN Bus communication
- Humidity and ultrasonic sensor data transmission
- Real-time LCD display using I2C
- CAT5 cable communication for longer distance simulation
- Smart elevator floor monitoring simulation

## Technologies Used
- Arduino Uno
- Arduino Nano
- MCP2515 CAN Bus Module
- I2C LCD
- Humidity Sensor
- Ultrasonic Sensor

## Future Improvements
- CAN Bus Audio Intercom System
