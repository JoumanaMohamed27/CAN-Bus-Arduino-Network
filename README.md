This project implements a multi-node communication system using Arduino Uno and Arduino Nano with the MCP2515 CAN Bus module. It simulates real-time embedded communication between multiple sensors and display units over a CAN network.

Features
Multi-node CAN Bus communication between Arduino boards
Transmission of humidity and ultrasonic sensor data over the network
Real-time data display using I2C LCD
Simulation of smart elevator floor monitoring system
Reliable long-distance communication using CAT5 cable
System Overview

The system consists of multiple nodes connected via a CAN Bus network:

Sensor nodes collect environmental and distance data
Data is transmitted through MCP2515 CAN modules
Receiver node processes and displays data on an LCD screen
Technologies Used
Arduino Uno
Arduino Nano
MCP2515 CAN Bus Module
I2C LCD Display
Humidity Sensor
Ultrasonic Sensor
